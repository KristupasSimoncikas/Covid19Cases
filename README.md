# Covid19Cases
Vue.js & .NET Core 3.1

Covid-19 cases management application

## Build

``` bash
cd Teltonika\Client
# To host front-end
npm run dev
# To host back-end
Don't forget to change database connections settings in application.json
Make sure applications runs on http://localhost:39710/
```
